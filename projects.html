JACI STICKROD

  • Cowboy Game
A single-player game using the pygame library to showcase OOP design principles.

For this project the task was to create a project that implemented at least 3 design patterns. My group wanted to make a cowboy themed game similar to the dinsosaur game (when your wifi is out). I generated multiple UML diagrams, ran unit tests and added code to support the design patterns we chose, Mediator, Adapter, Singleton and Decorator. Over the course of our project our patterns adapted to the functionality we wanted to implement. The pygame library was a great resource and allowed us to start running good code quickly. We made some of our assets as well as using free assets on the internet. It was great to use python for a easy game to demonstrate design princiles.
